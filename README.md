# Java Practice

Some simple Java syntax practice. Something to look back to for a refresh.

## Usage

For `1/message-printer`, compile and run the driver class `MessagePrinter`

For `2/calorie-intake`, compile and run the driver class `CalorieTracker`

For `3/race-simulator`, compile and run the driver class `RaceSimulator`

For `4/exam-attempt-marker`, compile and run the driver class `MarkExams`

For `5/aeroplane-simulator`, compile and run the driver class `FlightSimulation`
